# Web (React + Vite) — ES/EN

## Español

Frontend minimal para demo Full Stack.

### Run local

```bash
cp .env.example .env
npm install
npm run dev
```

### Deploy (Vercel)
- Importá el repo
- Root Directory: `web`
- Seteá env var `VITE_API_BASE_URL` con la URL de tu API (Render/Fly/etc)
